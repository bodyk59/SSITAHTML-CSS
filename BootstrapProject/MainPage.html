MindIT

MindIT is a program to create and edit database of employees of your company. Use appropriate tools to add, update, delete or simply review your employees’ database. Create a report for official and personal needs.

Documentation

Step 1

To login into MindIT for the first time, type in “root” in Username field and Password field. For the next time, proceed with your Department name as Username and Surname as a Password.

Next

Step 2

All necessary files are to be found in JavaProjectFXOOP folder.

Step 3

All plugins for successful operation including PostgreSQL Driver, ITEXT and JavaFX SDK 11.0.2 are stored in lib folder.
